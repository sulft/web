Apprentissage bootstrap 5

Section 1

1.Modifier la couleur de texte

Attention! Ne surtout pas considérer les crochets '[' ou ']' lors de l'écriture du code bootstrap !

Syntaxe : class="text-[couleur]"


Pour plus d'info sur les couleurs valides, cliquez ICI.

2.Modifier la couleur de "background"

Syntaxe : class="bg-[couleur]"


Pour plus d'info sur les couleurs valides, voir partie précédente.

3.Changer la longueur d'un élément

Syntaxe : class="w-[size]" ou "h-[size]"


Pour plus d'info sur les tailles(size) valides, cliquez ICI.

4.Changer le margin/padding d'un élément

Syntaxe : class="m-[value]" ou "p-[value]"


Pour plus d'info sur les margins/paddings valides, cliquez ICI.

Section 2

1.Aligner le texte

Syntaxe class="text-[start] ou [end] ou [top] ou [bottom]

Aligne le texte à l'emplacement concerné.

Note ! Start et end remplacement respectivement left et right

2.Transformer le texte

Syntaxe class="text-[uppercase] ou [lowercase] ou [capitalize]

Type Example
lowercase Je suis un test
uppercase Je suis un test
capitalize Je suis un test

3.Gestion des bordures

Syntaxe class="text-[uppercase] ou [lowercase] ou [capitalize]
